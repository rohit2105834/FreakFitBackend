FreakFit API Documentation

Authentication

POST /api/auth/login/

Authenticate user and receive an access token.

{ "username": "user123", "password": "mypassword" }

Users

GET /api/users/

Retrieve a list of users.

POST /api/users/

Create a new user.

{ "username": "newuser", "email": "user@example.com", "password": "mypassword" }

Workouts

GET /api/workouts/

Retrieve all available workouts.

POST /api/workouts/

Add a new workout.

{ "name": "Push-ups", "reps": 20, "sets": 3 }
